# 贡献说明

欢迎提交 Issue、建议和补丁。提交贡献前，请先确认你理解并接受本项目的授权模式：源码公开仅用于非商业用途，商业使用必须取得众汇云创科技（深圳）有限公司书面授权。

## 贡献授权

当你向本项目提交代码、文档、设计、脚本、配置或其他内容时，即表示：

1. 你有权提交该贡献，且该贡献不侵犯第三方权利。
2. 你同意将该贡献授权给众汇云创科技（深圳）有限公司用于本项目及其商业版本。
3. 你同意该贡献可按本项目当前或未来的授权模式进行使用、修改、分发和商业授权。
4. 你不会在贡献中提交密码、Token、密钥、私有证书、客户数据或其他敏感信息。

## 提交要求

1. 所有文档、注释和提交说明优先使用中文。
2. 代码必须遵守 `docs/standards/code-writing-standard.md`。
3. 涉及后端、前端、移动端、数据库、工作流、开放 API 或低代码生成时，必须执行匹配范围验证。
4. 不得提交 `node_modules`、`target`、`dist`、本地 `.env`、IDE 私有配置和临时文件。
5. 新增第三方依赖时，应同步检查许可证并更新 `THIRD-PARTY-NOTICES.md`。

## 安全问题

安全漏洞、密钥泄露、越权、租户隔离缺陷请不要直接公开到普通 Issue。请按 `SECURITY.md` 的方式私下报告。
